The Browser as an Application Platform

Modern browsers support installable, offline-capable experiences and near-native performance for demanding apps.

Progressive Web Apps (PWAs)

  • Installable with app manifests
  • Service Workers enable offline and background features
  • Expanding capabilities across engines (files, notifications, storage)

Performance Frontiers

  • WebAssembly (Wasm): compiled languages (C/C++/Rust) at near-native speed alongside JavaScript
  • WebGPU: high-performance graphics and compute for creation tools, ML inference, and simulation

Together with richer web APIs (files, payments, capture), the web now powers productivity suites, games, and AI-augmented tools without traditional installs.
